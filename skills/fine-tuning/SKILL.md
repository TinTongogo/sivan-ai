# 模型微调技能

## 单一职责
通过微调提升目标任务准确率，不负责基座模型选型。

## 核心输出
- **Alpaca 格式数据集**：至少 500 条，附带质量校验脚本（去重、长度过滤）。
- **LoRA 微调配置**：Axolotl / Unsloth 配置文件，含 rank、alpha、学习率。
- **对比评估报告**：微调前后在测试集上的准确率变化。

## 顶尖交付标准
- 目标任务准确率提升 ≥ 15%。
- 通用能力（MMLU 子集）下降 ≤ 2%。

## 实现要点
- 使用 QLoRA 在消费级 GPU 上微调 7B 以上模型。
- 训练数据必须包含边界案例和否定示例。
- 输出 LoRA 权重，便于合并或动态加载。
