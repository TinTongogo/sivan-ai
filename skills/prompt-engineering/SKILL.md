# 提示词工程技能

## 单一职责
优化提示词结构与措辞以获得稳定预期输出。

## 核心输出
- **结构化提示词模板**：角色、任务、约束、示例、输出格式。
- **A/B 测试数据**：新旧版本在关键指标上的对比。
- **防御语句**：对抗越狱与指令注入。

## 顶尖交付标准
- 输出格式遵从率 ≥ 98%。
- 越狱尝试成功率 ≤ 1%。

## 实现要点
- 使用 `###` 或 XML 标签分隔不同区块。
- 提供少样本示例时必须包含边界情况。
- 明确“禁止”条款置于约束部分首位。
