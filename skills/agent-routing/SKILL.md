# 智能体路由技能

## 单一职责
将输入文本映射到目标智能体名称。

## 核心输出
- **关键词匹配规则表**：各智能体对应的关键词集合。
- **相似度计算逻辑**：Jaccard 相似度 + 停用词过滤。
- **回退提示模板**：无法识别意图时的用户引导语。

## 顶尖交付标准
- 隐式路由准确率 ≥ 90%（基于用户手动纠正数据）。
- 显式路由 `@agent` 优先级最高，直接跳过意图分析。

## 实现要点
- 输入预处理：转小写、去标点。
- 相似度阈值 0.3，低于则返回“无法识别”。
- 避免路由到协调器自身。
