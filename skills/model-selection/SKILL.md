# 模型选型技能

## 单一职责
输出有数据支撑的模型推荐结论。

## 核心输出
- **加权打分矩阵**：能力、延迟、成本、合规、上下文长度等维度。
- **PoC 对比实验数据**：在真实任务上的表现。
- **ADR 格式决策记录**：包含选型理由与风险。

## 顶尖交付标准
- 选型结果在预算内满足 P99 延迟要求。
- 决策文档包含至少一个被放弃的主流模型及明确放弃原因。

## 实现要点
- 成本计算使用实际 Token 单价。
- PoC 测试集必须与生产任务分布一致。
- 合规性需考虑数据驻留与许可证。
